## Run Locally

Clone the project

```bash
  git clone https://github.com/empaliizer/portfolio-oak
```

Go to the project directory

```bash
  cd portfolio-oak
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Deployment

To deploy this project run

```bash
   npm install
```
```bash
   npm run dev
```
